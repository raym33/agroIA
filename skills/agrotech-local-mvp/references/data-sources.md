# Data sources

Prefer these sources first:

- `SIAM-IMIDA` for Murcia agroclimate and ET0
- `AEMET OpenData` for broader climate fallback
- `SIGPAC` for official parcel geometry
- `Sentinel-2` for NDVI and vegetation vigor
- `IP102` and `AgroPest-12` for visual experimentation
- `FAO AquaCrop` as a physical-model reference

Rules:

- keep source attribution explicit
- check licensing before commercial dataset use
- do not imply field validation just because a public dataset exists
